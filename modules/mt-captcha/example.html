MT Captcha¶

To import this module:

from python3_captchaai.mt_captcha import MtCaptcha
class mt_captcha.MtCaptcha(api_key: str, websiteURL: str, websiteKey: str, proxy: str, sleep_time: Optional[int] = 5, request_url: Optional[str] = 'https://api.capsolver.com')¶

The class is used to work with Capsolver MtCaptcha method.

Parameters:

  • api_key (str) – Capsolver API key

  • websiteURL (str) – Address of a webpage with mtcaptcha

  • websiteKey (str) – sk=MTPublic-xxx public key

  • proxy (str) – String with proxy connection params, example: 198.22.3.1:10001:user:pwd

  • sleep_time (Optional[int]) – The waiting time between requests to get the result of the Captcha

  • request_url (Optional[str]) – API address for sending requests
