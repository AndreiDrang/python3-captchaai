Fun Captcha¶

To import this module:

from python3_captchaai.fun_captcha import FunCaptcha
class fun_captcha.FunCaptcha(api_key: str, captcha_type: Union[CaptchaTypeEnm, str], websiteURL: Optional[str] = None, websitePublicKey: Optional[str] = None, funcaptchaApiJSSubdomain: Optional[str] = None, proxyType: Optional[Union[ProxyType, str]] = None, proxyAddress: Optional[str] = None, proxyPort: Optional[int] = None, sleep_time: Optional[int] = 5, request_url: Optional[str] = 'https://api.capsolver.com')¶

The class is used to work with Capsolver FuncaptchaTask methods.

Parameters:

  • api_key (str) – Capsolver API key

  • captcha_type (Union[CaptchaTypeEnm, str]) – Captcha type name, like GeetestTaskProxyless and etc.

  • websiteURL (Optional[str]) – Address of a webpage with Geetest.

  • websitePublicKey (Optional[str]) – Funcaptcha website key.

  • funcaptchaApiJSSubdomain (Optional[str]) – A special subdomain of funcaptcha.com,from which the JS captcha should be loaded. Most FunCaptcha installations work from shared domains.

  • proxyType (Optional[Union[ProxyType, str]]) – Type of the proxy

  • proxyAddress (Optional[str]) – Proxy IP address IPv4/IPv6. Not allowed to use: host names instead of IPs, transparent proxies (where client IP is visible), proxies from local networks (192.., 10.., 127…)

  • proxyPort (Optional[int]) – Proxy port.

  • sleep_time (Optional[int]) – The waiting time between requests to get the result of the Captcha

  • request_url (Optional[str]) – API address for sending requests

captcha_handler(image: Optional[str] = None, question: Optional[str] = None, **additional_params) → CaptchaResponseSer¶

Synchronous method for captcha solving

Parameters:

  • image (Optional[str]) – Base64 encoded image, can be a screenshot (pass only the hexagonal image, do not pass the rest of the content)

  • question (Optional[str]) – Question name. Pass the full name, such as: Pick the lion

  • additional_params – Some additional parameters that will be used in creating the task and will be passed to the payload under task key. Like data, proxyLogin and etc. - more info in service docs
